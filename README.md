# COSC 650: Applied LLM Systems

This repository contains my coursework, notebooks, experiments, and project materials for **COSC 650: Applied LLM Systems** at Maryville University.

## Course Context

COSC 650 is an 8-week graduate course focused on understanding how large language models work as applied systems. The course covers the full LLM workflow, including tokenization, transformer-based inference, prompt engineering, function calling, retrieval-augmented generation, fine-tuning, evaluation, and responsible use of LLM tools.

This repository will serve as the central workspace for assignments, notes, code experiments, Google Colab notebooks, and the final course project.
